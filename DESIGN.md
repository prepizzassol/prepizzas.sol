## 1. Concepto Visual
- **Estilo:** Scrapbook / Artesanal / Watercolor.
- **Atmósfera:** Cálida, cercana, hogareña y profesional.
- **Elementos clave:** Bordes irregulares, texturas de papel, ilustraciones tipo "doodle" (garabatos) y marcos tipo Polaroid.

## 2. Sistema de Diseño (Design Tokens)

### Paleta de Colores
- **Background Principal:** `#FCF8F1` (Crema suave / Papel pergamino).
- **Acentos Watercolor (Pasteles):**
  - Naranja (Hero/Empanadas): `#FFDAB9`
  - Verde (Pizzas/Prepizzas): `#D4EDDA`
  - Lavanda (Personal/Info): `#E2D1F9`
  - Amarillo (Detalles/Botones): `#FFF3CD`
- **Tipografía (Color):**
  - Títulos/Cuerpo: `#4A3728` (Marrón café profundo para evitar el negro puro).

### Tipografía sugerida (Google Fonts)
- **Títulos (H1, H2, H3):** `Handlee` o `Indie Flower` (Estilo manuscrito/handwritten).
- **Cuerpo de texto:** `Quicksand` o `Nunito` (Sans-serif redondeada y amigable).

## 3. Detalles de Interacción & UI
- **Hover Effects:** Las imágenes tipo Polaroid deben "enderezarse" ligeramente al pasar el mouse (`transform: rotate(0deg)`).
- **Texturas:** Aplicar un sutil `background-image` de grano de papel sobre el fondo crema para dar profundidad.
- **Iconografía:** No usar íconos vectoriales estándar (FontAwesome/Lucide). Usar ilustraciones hechas a mano o SVGs con trazos irregulares.
